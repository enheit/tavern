### S6.3 local update roundtrip — 2026-07-09 — DoD: install → bump patch → manifest to local R2 → update applies

Setup: both versions built with a --config overlay pointing the updater at the local
worker (endpoints=["http://localhost:8787/updates/latest.json"] +
dangerousInsecureTransportProtocol, needed only because the local worker is plain
http — prod stays https). Signed with the real key (~/.tauri/tavern.key). Repo files
untouched (tauri.conf.json stays 0.1.0 / prod endpoint).

$ pnpm tauri build --config update-local-0.1.0.json --bundles app   # TAURI_SIGNING_PRIVATE_KEY set
$ ditto target/release/bundle/macos/Tavern.app /Applications/Tavern.app
$ defaults read /Applications/Tavern.app/Contents/Info.plist CFBundleShortVersionString
0.1.0

$ pnpm tauri build --config update-local-0.1.1.json --bundles app   # overlay: version=0.1.1
$ node worker/scripts/gen-update-manifest.ts --version 0.1.1 \
    --notes "S6.3 local update roundtrip test" --base http://localhost:8787 \
    --artifact darwin-aarch64:Tavern.app.tar.gz:Tavern.app.tar.gz.sig -o latest.json
$ npx wrangler r2 object put tavern-updates/Tavern.app.tar.gz --file ... --local
$ npx wrangler r2 object put tavern-updates/latest.json --file ... --local

$ curl -s http://localhost:8787/updates/latest.json        # wrangler dev, local R2
{"version":"0.1.1","notes":"S6.3 local update roundtrip test",
 "pub_date":"2026-07-09T01:52:31.671Z","platforms":{"darwin-aarch64":{
 "signature":"<minisign sig>","url":"http://localhost:8787/updates/Tavern.app.tar.gz"}}}

$ /Applications/Tavern.app/Contents/MacOS/tavern-desktop    # launch installed 0.1.0
[updater] downloading 0.1.1
[updater] installed 0.1.1 (applies on relaunch)

$ defaults read /Applications/Tavern.app/Contents/Info.plist CFBundleShortVersionString
